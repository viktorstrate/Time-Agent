# Time Agent

![](./screenshot.png)

## Features

- Easy one-click timer
- Organize projects in groups
- Archive tasks
- Export projects to PDF, CSV or HTML
- Synchronize projects to a file, to share projects across multiple computers with cloud services like Dropbox
